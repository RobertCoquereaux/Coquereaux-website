Hardware
You need:
- An Ivory board (I have a MacIvory model II)
- A Mac II FX with at least two empty NuBus slots, in order to plug the Ivory board (a Mac II is OK, better if you have a II FX or a Quadra).
- At least one SCSI hard disk attached to the FX
- A Mac G4 or G5 (I did not try the recent ones with Intel Core)
- Ethernet cards in the FX  Nubus: at least one is needed, better if you have two. I shall suppose that you have two of them.
- One ethernet connexion between the FX and the G5. In my case, the ethernet cable goes to a hub. From the hub it goes to an Airport base, and the G5 is connected to the base by wifi but this is irrelevant.
- One ethernet cable connecting the two ethernet cards on the NuBus FX board (one ethernet is used by  the Ivory and the other by the FX host)
 
Mac II FX Software
 Any system 7 would have worked for our purposes, in the past, but  Apple changed the communication protocol and older machines can no longer communicate simply with new ones. You need to install ShareWayIP on the Mac II side if you want its disks to be seen by the G5. SharewayIP is not free but is not very costly. Check the web. The point is that it requires system 7.5.5 to work. I had 7.5.3 so I had to find a more recent version (but not too recent because the communication between the Ivory and the Mac II would not work with a system higher than 8).
Strictly speaking, if the only purpose of the game is to send a X Window from the Ivory to the G5, one does not need to mount the FX hard disk on the G5 desktop... But it is much better if you can do it.
So you need to:
- Get and install ShareWayIP on the Mac II FX.
- Install system 7.5.5
 
Of course you also need
- The MacIvory Control Panel (it comes with the Symbolics distribution)
- The TCP/IP Control Panel (it comes with Apple system 7.5.5)
- The genera mac application (that will launch the FEP = Front End Processor of the Ivory machine)
 
For backup reasons (explained later) it is a good idea to have a copy of the old Resedit application.
 
 
Ivory Software :
The Genera files : I use version 8.3.
Ivory files are resident in one or several  invisible partitions of your FX hard disk. They are called FEP partitions: FEP0, FEP1, FEP2...  
In my case : Fep0 contains boot files and basic genera worlds, Fep1 is broken, Fep2 contains the LMFS (Lisp Machine File Syetem), FEP3 contains a copy of the old Symbolics distribution and Fep4 contains my configured worlds.
Fep partitions are created by the MacIvory Control Panel. Once created they are initialized by the Create Initial Filesystem command.  The same Control Panel can also copy the partitions, for backup purposes for instance,  but you have then to make the backup on a SCSI disk... which is something difficult to find these days. So the idea is to make a backup in another way : via the network. I shall therefore digress a bit to explain how this can be down nowadays.
