From a    
Mac Ivory Lisp Machine
to a
 G4 or G5 Apple computer running OSX Tiger
via X11
 
The purpose of these notes is to explain how to send X11 windows from a Symbolics Ivory Lisp Machine to a recent G4 or G5 Apple computer running OSX Tiger.
I strongly doubt that anybody on Earth will use these notes since the number of MacIvory boards still operational is probably very small !  
So, I write these notes for myself, just in order not to forget how I did it !
